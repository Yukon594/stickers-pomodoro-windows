# 贴纸番茄钟 (Sticker Pomodoro)

<p align="center">
  <img src="Icon.png" alt="贴纸番茄钟图标" width="128" />
</p>

一个基于 Tauri 2 构建的可爱风 macOS 桌面番茄钟应用 —— 上传小人头像，专注结束时从屏幕角落弹出贴纸提醒。

## 概览

贴纸番茄钟是一款运行在 macOS 菜单栏的番茄工作法计时器。它把计时、任务、统计打包在一个紧凑的贴纸风格窗口中，通过手绘树苗、自定义贴纸和 iMessage 风格的提醒弹窗，让专注这件事变得轻软可触。

## 功能模块

### 计时系统
- **两种模式**：倒计时（25 分钟专注 + 5 分钟休息，可调）与正计时（自由计时并自动按周期分割为"树"）
- **圆形拨盘**：主界面中央的圆环既是计时显示器也是开始/暂停按钮
- **快速开始**：预设四种场景（读书、健身、写作、整理），一键启动对应时长的专注
- **全局快捷键**：`Ctrl + Alt + P`（可自定义），后台直接开始或恢复番茄，不用切回主窗口

### 提醒弹窗
- **macOS 原生风格**：透明背景 + 毛玻璃卡片，位于屏幕顶部（左上 / 顶部中央 / 右上可选）
- **操作按钮**：专注结束时弹出"休息一下"和"下一棵树"按钮，点击即可切换状态
- **持久化提醒**：带有操作按钮的提醒不会自动消失，直到用户点击
- **自定义文案**：四类提醒（专注开始/结束、休息开始/结束）的标题和正文均可自定义
- **系统声音**：14 种 macOS 原生提醒音效可选，支持触觉反馈

### 菜单栏小树
- **5 种树样式**：圆润幼苗、像素小树、小盆栽（默认）、松树剪影、手账印章
- **动态生长**：树苗随倒计时进度从 stage 0 长到 stage 4
- **休息状态**：休息倒计时期间图标切换为充电电池样式
- **Retina 适配**：18pt 逻辑画布 + 高分辨率位图渲染

### 项目管理
- **多项目支持**：创建、重命名、删除项目，8 种颜色标记
- **项目归类**：Todolist 和森林统计均按项目归类
- **未分类转移**：专注完成后可将未分类的会话转移到指定项目

### Todolist

<p align="center">
  <img src="%E5%9B%BE%E7%89%87/%E2%80%9C%E4%BB%8A%E6%97%A5%E2%80%9D%E6%9D%BF%E5%9D%97.png" alt="今日板块" width="600" />
</p>

- **今日待办**：添加、编辑、完成、删除待办事项
- **计划时间**：每项待办可设置预计专注分钟数
- **拖拽排序**：支持指针拖拽调整顺序
- **进度追踪**：记录每项待办的专注秒数和完成树数
- **已完成列表**：当日已完成的待办可撤回

### 贴纸系统
- **背景贴纸**：上传 PNG/JPEG/WebP/GIF 图片，自由拖动、缩放、旋转、镜像翻转
- **贴纸布置模式**：专用编辑模式，提供旋转、翻转、删除工具条

### 自定义背景
- **上传背景图**：支持铺满/完整两种显示方式，可调节透明度
- **自动配色匹配**：从背景图提取主色调，自动调整界面配色以保证可读性

### 森林统计

- **12 周热力图**：类似 GitHub 贡献图的专注热力图
- **两种指标**：按"树数"或"专注时长"查看
- **项目筛选**：按项目过滤统计数据
- **三级汇总**：今日 / 本周 / 本月 的树数和专注时长

- **4 种预设主题**：奶油、薄荷、樱桃、夜晚，可跟随系统深浅色模式
- **提醒位置**：左上 / 顶部中央 / 右上，切换时实时预览
- **全局快捷键**：可自定义组合键，后台直接开始或恢复番茄
- **自定义文案**：四类提醒的标题和正文均可修改
- **菜单栏开关**：可切换树样式，实时预览生长效果

## 技术架构

```
src/
├── App.tsx                 # 主界面 + 设置面板 + 提醒弹窗
├── components/
│   └── ErrorBoundary.tsx   # React 错误边界
├── hooks/
│   ├── useTimer.ts         # 计时器状态管理
│   ├── useSettings.ts      # 设置读写 + 持久化
│   ├── useTodos.ts         # Todolist 状态管理
│   ├── useProjects.ts      # 项目状态管理
│   └── useStickers.ts      # 贴纸状态管理
├── lib/
│   ├── types.ts            # TypeScript 类型定义
│   ├── timer.ts            # 计时器核心逻辑
│   ├── storage.ts          # 本地持久化 + Tauri 桥接
│   ├── trayForest.ts       # 菜单栏图标渲染 (Canvas → PNG → Tauri)
│   ├── stats.ts            # 森林统计 + 热力图数据
│   ├── todos.ts            # Todolist 业务逻辑
│   ├── audio.ts            # Web Audio 音效反馈
│   └── palette.ts          # 背景色自动提取
├── styles.css              # 视觉系统 + 主题 + 响应式
└── main.tsx                # 入口

src-tauri/
├── src/main.rs             # Tauri 后端：托盘、提醒窗口、快捷键、触觉反馈
├── tauri.conf.json         # Tauri 2 配置（无边框透明窗口 560×620）
└── icons/                  # 应用图标
```

- **前端**：React 19 + TypeScript + Vite 6，`lucide-react` 图标库
- **桌面框架**：Tauri 2，Rust 后端处理系统级功能
- **窗口特性**：无边框透明窗口、关闭即隐藏、自定义红绿灯控件
- **数据持久化**：JSON 文件存储，localStorage 兜底
- **测试**：Vitest，覆盖计时器、统计、Todolist、持久化核心逻辑

## 下载安装

[![GitHub Release](https://img.shields.io/badge/Download-v1.1.0-blue)](https://github.com/Yukon594/stickers-pomodoro/releases/latest)

从 [Releases](https://github.com/Yukon594/stickers-pomodoro/releases) 页面下载最新的 `.dmg` 文件，将 `贴纸番茄钟.app` 拖入 `应用程序` 文件夹。

应用未经过 Apple 开发者证书签名，首次打开前需在终端执行以下命令解除隔离：

```bash
xattr -d com.apple.quarantine /Applications/贴纸番茄钟.app
```

之后即可正常双击打开，后续启动不会再被拦截。仅支持 macOS (Apple Silicon)。

## 开发运行

```bash
# 安装依赖
npm install

# 浏览器开发（http://127.0.0.1:1420/）
npm run dev

# Tauri 桌面开发（需要 Rust 工具链）
npm run tauri dev

# 类型检查 + Vite 构建
npm run build

# 运行测试
npm test

# 桌面应用打包
npm run tauri build
```

## 配置参数

| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| 专注时间 | 25 分钟 | 可调范围 1-60 分钟 |
| 休息时间 | 5 分钟 | 可调范围 1-30 分钟 |
| 提醒位置 | 右上 | 左上 / 顶部中央 / 右上 |
| 提醒声音 | Blow | 14 种 macOS 系统声音 |
| 菜单栏树样式 | 小盆栽 | 5 种可选 |
| 全局快捷键 | Ctrl+Alt+P | 可自定义组合键 |
| 主题 | 奶油 | 跟随系统外观可选 |
| 背景自动匹配 | 开启 | 从背景提取配色方案 |

## 设计原则

- **贴纸感**：圆角、柔影、低饱和色彩，界面紧凑不拥挤
- **直给操作**：主拨盘就是开始/暂停按钮，不需要多步导航
- **本地优先**：所有用户数据（头像、贴纸、背景）存储在本地，不上传
- **关闭即隐藏**：主窗口关闭后隐藏到菜单栏，不退出应用
- **原生感提醒**：提醒窗口透明无边框，行为贴近 macOS 原生通知
