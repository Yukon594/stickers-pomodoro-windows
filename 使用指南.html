贴纸番茄钟 · 使用指南

可爱风的 macOS 番茄工作法计时器 — 种树、打卡、放轻松

v1.1.0 · 仅支持 macOS

📥 下载与安装

  1. 下载 DMG
    前往 GitHub Releases 页面， 下载 贴纸番茄钟_x.x.x_aarch64.dmg。
  2. 拖入应用程序
    打开 DMG，将 贴纸番茄钟.app 拖入 应用程序 文件夹。
  3. 解除 macOS 隔离
    应用未签名，首次使用前在终端粘贴运行： 
    xattr -d com.apple.quarantine /Applications/贴纸番茄钟.app
  4. 双击打开
    之后就可以正常双击启动了。后续再打开不会被拦截。

🏠 认识主界面

左上角

个人头像 + 工具栏（今日、统计、设置）。点击头像可以上传自己的小人图。

中央拨盘

红色的圆形区域就是开始/暂停按钮。外圈显示剩余时间，进度环随计时增长。

窗口控制

左上角的红绿灯按钮：关闭 = 隐藏到菜单栏（不退出），黄色 = 最小化，绿色 = 缩放。

⏱ 开始专注

方式一：主界面按钮

  1. 选择计时模式：倒计时（预设 25 分钟）或 正计时（不限时）
  2. 打开项目菜单，选择本次专注归属的项目
  3. 点击中央圆盘开始 — 窗口自动最小化，菜单栏树苗开始生长
  4. 计时结束弹出提醒，可选择「休息一下」或直接开启「下一棵树」

方式二：快速开始

打开「今日」面板，顶部的快速开始卡片预设了读书、健身、写作、整理四种场景。
点一下直接启动，不用再调时间、选项目。每个卡片的时间、名称和归属项目都可以编辑。

方式三：全局快捷键

默认 Ctrl + Alt + P。任何时候按下 — 即使贴纸番茄钟在后台 — 都会立即开始一轮 25 分钟专注。
快捷键可以在设置里自定义，也可以关闭。

💡 提示：专注中再次按快捷键不会暂停。如果要暂停或跳过，请在菜单栏右键图标操作。

休息与跳过

每轮专注结束默认接 5 分钟休息。休息期间菜单栏图标会变成充电电池样式。
不想休息？点击 跳过 按钮直接开始下一轮。也可以点 重置 回到初始状态。

📋 今日待办 (Todolist)

  1. 在底部输入框填写任务名、预计分钟数、归属项目，点击 + 添加
  2. 点击任务行选中，然后中央拨盘下方会显示当前任务标签
  3. 专注结束后，该任务自动累计专注秒数和树数
  4. 打勾完成的任务移到「Done list」，可以撤回
  5. 长按拖拽可以调整任务顺序
  6. 双击任务名可以编辑标题

🌲 森林统计

点击主界面的「统计」按钮打开：

热力图

12 周热度方块，类似 GitHub 贡献图。颜色越深表示当天专注越多。

切换指标

「树」= 完成轮数，「时长」= 总专注分钟。按项目筛选只看某个分类。

三级汇总

今日 / 本周 / 本月 的树数和时长一目了然。

⚙ 个性化设置

🎨 主题

4 种配色：奶油（默认）、薄荷、樱桃、夜晚。开启「跟随系统」自动随深浅色切换。

🖼 背景

上传自定义背景图，铺满或完整显示，调节透明度。打开「自动匹配」会从图片提取配色。

🔔 提醒

选择弹窗位置（左上/上中/右上）。14 种系统提示音可选。自定义四类提醒文案。

⌨ 快捷键

点击快捷键按钮进入捕获模式，按下组合键即可设定。默认 Ctrl+Alt+P。

🌱 菜单栏树

5 种树样式：圆润幼苗、像素小树、小盆栽（默认）、松树剪影、手账印章。

📂 项目管理

创建、重命名、删除项目，8 种颜色可选。删除项目需要确认。

🏷 贴纸布置

上传图片作为背景贴纸，自由拖拽、旋转、翻转、缩放。最多 24 张。

👤 头像

上传小人头像，会出现在提醒弹窗中。

🍃 菜单栏操作

贴纸番茄钟常驻 macOS 菜单栏。右键（或左键）图标弹出菜单：

菜单项作用
打开贴纸番茄钟显示主窗口
开始 / 暂停切换计时状态
重置重置当前计时
查看森林统计打开统计面板
退出完全退出应用

菜单栏图标本身也会显示剩余时间和今日树数。按住 ⌘Command 可以拖动图标调整位置。

💡 如果图标不显示：检查是否有 Bartender、Hidden Bar 等第三方工具隐藏了它；也可尝试重启菜单栏：killall SystemUIServer。

❓ 常见问题

窗口关了怎么重新打开？

点击菜单栏的树图标，选择「打开贴纸番茄钟」。关闭按钮只是隐藏，不会退出应用。

正计时和倒计时有什么区别？

倒计时是设定好的番茄钟（默认 25 分钟专注 + 5 分钟休息）；正计时是自由计时，每满一个周期自动算一棵树。

数据保存在哪里？

所有数据（头像、贴纸、背景、统计）都以 JSON 文件存在本地 ~/Library/Application Support/com.stickerpomodoro.timer/，不会上传到任何服务器。

如何更新到新版本？

去 GitHub Releases 下载最新 DMG，替换旧的 .app 即可，设置数据会自动迁移。
